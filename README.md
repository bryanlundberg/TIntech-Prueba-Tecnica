# Prueba técnica

  Para este ejercicio,  se utilizaron las siguientes tecnologías, de la mano de la API de [Last.FM](https://www.last.fm/es/api)

- React
- Nextjs
- MongoDB
- Tailwind
- pnpm

Las variables de entorno necesarias en el archivo `.env`

    MONGODB_URI=
    LAST_FM_BASE_URL=
    LAST_FM_API_KEY=
    JWT_SECRET_KEY=
